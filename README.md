# smolx2

<img width="1710" height="960" alt="image" src="https://github.com/user-attachments/assets/2de3fd85-2751-4f73-b4f6-5a4f26afb71f" />


A game where you play connect four with rings, but some of the rings have magic gemstones that can destroy and swap with other rings! Be careful, because if you don't use a magic gemstone, it will eventually break.
play it here: https://that-blob.itch.io/smolx2
<img width="1710" height="960" alt="image" src="https://github.com/user-attachments/assets/b74b35a3-a72a-4100-9c99-3718441e8ddc" />

This was made with Godot for code, Aseprite for art, and Chiptone for music!

<img width="1710" height="960" alt="image" src="https://github.com/user-attachments/assets/cd916339-f611-4d1a-b7ad-1d9ce81aff1d" />
